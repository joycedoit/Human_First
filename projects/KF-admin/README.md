# 專案名稱：當行政遇到 AI — 從日常協作到流程自動化的導入經驗與反思

## 基本資訊
- **團隊／負責人**：Joyce（行政）
- **聯絡方式**：joyce@sky-net.tw
- **一句話介紹**：行政導入 AI 的四個真實經驗：從「做一件事要再做一次」，到讓工作可追蹤、可移交、可自動跑。

## 這個專案在做什麼
- **解決的問題 / 痛點**：行政工作重複性高、無法標準化、人走知識就消失
- **使用者 / 適用情境**：行政、HR、非技術背景人員導入 AI 自動化流程

## 用了哪些 AI 工具與技術
- **AI 工具 / 模型**：ChatGPT（發想）、Claude Code（程式碼生成與精煉）
- **框架 / 技術棧**：n8n（流程自動化）、Render（雲端部署）、GitHub（工作流追蹤）
- **可重用的 SKILL / SCRIPT / prompt**：無（本次分享以經驗為主）

## 成果與效益
- 行政大腦：GitHub + n8n 讓內部任務全透明，自動雙向通知＋任務提醒
- 計薪自動化：耗時從 3–4 天縮短至 10 分鐘內
- 流程封裝：複雜 API 操作包成一鍵介面，團隊成員「無腦維護及使用」
- 活動創意：技術門檻消失後，即時投票、母親節特輯、卡片生成器都自己做

## 架構簡述
AI 應用分四個層次：內容生成 → 資料整理 → 流程自動化 → 管理應用（行政大腦）。
工具鏈：ChatGPT 發想 → Claude Code 生成程式 → n8n 串接自動化 → Render 雲端部署。
